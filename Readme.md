# Personal Portfolio Website

A multi-page personal portfolio website built using **pure HTML** as part of the **Nexus Front-End Boot Camp Project**.
This project demonstrates fundamental web development concepts including semantic HTML, multi-page navigation, forms, tables, lists, and proper project structure.

---

## 📌 Project Overview

This portfolio introduces **Amanuel Mebratu**

The website provides information about:

* Personal background
* Technical skills
* Engineering interests
* Learning goals
* Contact information

The project is built **using only HTML** to demonstrate a strong understanding of core web structure without CSS or JavaScript.

---

## 🚀 Features

* Multi-page website structure
* Semantic HTML elements
* Internal page navigation
* External social links
* Clickable images
* Lists and tables
* Contact form with multiple input types
* Organized project folder structure

---

## 🧱 Technologies Used

* HTML5

(No CSS or JavaScript used as required by the project guidelines.)
